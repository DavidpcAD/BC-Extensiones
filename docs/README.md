# Documentacion del proyecto

## Estructura

- `api/`: documentacion tecnica de APIs y endpoints.
- `powerapps/`: ejemplos y snippets para consumo desde Power Apps.
- `guides/`: guias funcionales, troubleshooting y changelog.

## Inicio rapido

1. Leer `guides/DOCUMENTACION-COMPLETA.md` para vista general.
2. Ir a `api/` segun endpoint.
3. Tomar ejemplos de `powerapps/` para integracion.
